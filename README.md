# ReactJs + Hooks + Redux + Nodejs - Building a Blog with MaterialUI

## Requirements
* **nodejs >= v14.17.2**
* **npm >= v6.14.13**
* **yarn >= v1.22.11**


Clone project and run test.
```
$ git clone --recurse-submodules https://github.com/vinhqngo5/MERN-Blog.git
$ cd MERN-Blog/server
$ yarn install
$ yarn start
$ ctrl + shift + T or ctrl + shift + `
$ cd ../client
$ yarn install
$ yarn start
```
